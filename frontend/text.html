Главный заголовок

Это начало нашей веб-страницы. Здесь мы использовали тег <h1> для создания главного заголовка страницы.

Заголовок раздела

Этот заголовок отмечает начало нового раздела контента. Мы использовали тег <h2>. В этом абзаце мы демонстрируем обычный текст. Мы используем тег <p> для создания абзацев. Список
      - Первый элемент списка
      - Второй элемент списка
      - Третий элемент списка
Здесь мы создали маркированный список с помощью тега <ul>. Каждый элемент списка создан с использованием тега <li>. Заголовок изображения внутри раздела Выше вы видите изображение. Мы вставили его с помощью тега <img>. Специальный текст Мы можем выделить текст каким-то особым образом, используя тег <em>. Жирный текст Мы можем сделать текст жирным с помощью тега <strong>. Это всего лишь небольшой обзор HTML-тегов, которые вы можете использовать для структурирования и форматирования текста на веб-странице.
